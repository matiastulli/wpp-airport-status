# wpp-airport-status

A WhatsApp-powered airport status service that provides real-time flight and airport information via WhatsApp messaging.

## Features

- Real-time airport status updates
- Flight information lookup
- WhatsApp integration for easy access

## Getting Started

### Prerequisites

- Python
- A WhatsApp Business API account or WPP Connect setup

## Usage

Send a WhatsApp message to the configured number with an airport IATA code (e.g., `JFK`, `LHR`) to receive the current status.
